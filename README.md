# Age Bold — Playground

Design and prototype explorations for the Age Bold platform.

**Live site:** [agebold.github.io/playground](https://agebold.github.io/playground/)

## Projects

| Folder | What's inside |
|---|---|
| `dynamic-header/` | App prototype — today plan, class, onboarding, navigation |
| `MSK-ACCESS/` | 28-screen patient journey for MSK care access and onboarding |
| `care_plan/` | Care plan screens, MSK progress tracking and program views |
| `telehealth/` | Telehealth video call interface and flow |
| `design-system/` | Tokens, type scale, spacing, and component library |
| `user-research/` | Card sort and tree test tools |
| `vision/` | Product vision and strategy overview |

## How the index works

The index page is auto-generated from `projects.json` on every deploy. Don't edit `index.html` by hand — changes will be overwritten. To update the index locally:

```
npm run generate
```
